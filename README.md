Café Sales Dataset — Data Cleaning

📌 Project Overview

This project demonstrates how to clean and transform a raw café sales dataset using Python (pandas) in JupyterLab. The dataset contained missing values, inconsistent entries (ERROR, UNKNOWN), and mixed data types. Through systematic data wrangling, the dataset was refined into a structured and reliable format suitable for analysis and reporting.

🔑 Key Objectives

Remove duplicates and invalid entries (ERROR, UNKNOWN).

Handle missing values across multiple columns.

Convert data types for consistency:

Quantity → integer

Price Per Unit & Total Spent → float

Transaction Date → datetime

Sort transactions by Transaction ID for readability.

Export the cleaned dataset to Excel (Cafe_Sales_Cleaned.xlsx).

📂 Repository Structure

uncleaned_cafe_sales.csv → Raw dataset with errors and missing values.

Cafe_Sales_Dataset_Cleaned - JupyterLab.pdf → Jupyter notebook workflow documenting the cleaning process.

Cafe_Sales_Cleaned.xlsx → Final cleaned dataset ready for analysis.

🛠️ Tools & Libraries

Python
